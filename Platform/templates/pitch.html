Something broke and it ain't working, what do I do?

We are a self sufficient community over here (or we pretend to be atleast). Follow these basic debugging steps to try and solve it:

- Find the problem on popular forums & social sites (Reddit, Stack Overflow, Discord, etc) and see if a solution presents itself
- Check the devlogs to see if the bug has been listed, if it has a fix & whats causing it
- Still ain't working? Damn, message me (deets in the about section) about it and I will do my best to include it in the next patch
- Nuclear Option - If it appears that the issue is frontend, install TamperMonkey and modify the website yourself
